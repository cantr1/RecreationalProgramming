# Recreational Programing
This repo is where I like to keep things I have built that do not really mesh well with a professional portfolio.

Whether it be unconventional languages, or silly scripts that are just for fun, this repo is for keeping these kinds of things.

### C_Shell
The classic project for newbie C developers - a recreation of the UNIX shell written in C.

### C_Valut
An experiment with APIs in C - a project for a password vault that allows the user to use CRUD operations over the wire to store
their passwords (albeit insecurely, lol)

### ChessPy
A recreation of the classic game in Python played in the terminal. The main purpose besides flexing programming muscle was
to study the Strategy design pattern.
